### Strikeout

A `strikeout` annotation is used to draw a line through text.

```js
{
  "class": "Annotation",
  "type": "strikeout"
  "page": 1,
  "uuid": "ad9fe5b8-699d-4711-a94e-4b0eb02e551f",
  "color": "FF0000",
  "rectangles": [
    {
      "height": 12,
      "width": 457,
      "x": 72,
      "y": 147
    },
    {
      "height": 12,
      "width": 427,
      "x": 72,
      "y": 163
    }
  ]
}
```
